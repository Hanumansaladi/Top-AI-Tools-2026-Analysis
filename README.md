# Top AI Tools 2026 Analysis

## Project Overview
This project analyzes the top AI tools in 2026 using Python and Exploratory Data Analysis (EDA).

## Dataset Features
- Tool Name
- Primary Category
- Sub Category
- Pricing Model
- Starting Price (USD)
- Monthly Traffic Estimate
- User Rating
- API Availability

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Tasks
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Insights on AI tool trends

## Goal
To understand trends in AI tools such as category popularity, pricing models, and API availability.
